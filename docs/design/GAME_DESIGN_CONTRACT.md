# 고양이 오디세이 — Game Design Contract

## 제작 모드

원샷 제작. 추가 선택 질문 없이 구현·QA·패키징까지 진행한다.

## 한 줄 설명

둥근 신화 고양이의 전신을 당겨 쏘고, 벽·아군·괴수 부위에 정확히 반사시켜 오디세이의 열 항로를 돌파하는 세로형 플릭 리코셰 액션 RPG.

## 핵심 재미

- 예상 궤적과 실제 이동이 같은 결정론적 판정에서 오는 신뢰.
- 좁은 약점, 방향 방패, 아군 접촉 순서를 한 발에 잇는 각도 계산.
- 캐릭터마다 다른 반사·관통·중량·폭발 특성으로 같은 경기장을 다르게 푸는 편성.
- 짧은 한 발의 충돌, 히트스톱, 카메라 충격, 연계기와 부위 파괴가 만드는 손맛.

## 핵심 루프

`항구 → 항로/스테이지 선택 → 1~4마리 편성 → 플릭 전투 → 가호/전리품 → 강화·소환 → 재출정`

1·2·3마리 편성도 모든 캠페인 스테이지를 클리어할 수 있어야 하며, 특정 영웅이나 세 속성을 강제하는 승리 조건에는 소수 편성 대체 규칙을 둔다.

## 1분 경험

첫 발사로 벽 반사와 약점 타격을 경험하고, 첫 승리 보상과 캐릭터 성장 선택을 본다.

## 5분 경험

아군 연계, 방향 방어 적, 보스 부위 또는 항로 고유 물리 중 하나가 추가되고, 편성 또는 가호 선택이 결과를 바꾼다.

## 장기 목표

- 메인 10항로 43스테이지 완주.
- 영웅 수집·각성·동료도·도감 완성.
- 아포나의 신탁탑 30층.
- 주간 시드형 포-사이돈 폭풍 항로 기록 갱신.
- 12마리를 요구하는 3파티 신화 괴수 토벌.

## 후반 이야기 연결

- EP11 이후 Route 7~10은 새 영상을 필수로 만들지 않고 정사 인게임 카드로 원전 인과를 잇는다.
- 저승 귀환과 퍼-씨의 경고 → 사이렌 → 두 재앙 사이의 선택 → 트리나키아의 금기 → 난파와 오기기아 표류 → 파이아키아 회상 복귀 → 이타-캣 변장 귀환 순서를 유지한다.
- Route 10 최종전 뒤에는 침상의 비밀과 가족 재회를 담은 종장 카드가 있다.
- 영상·카드가 실패하거나 스킵되어도 다음 스테이지와 보상 흐름은 막히지 않는다.

## 실패와 복구

- 파티 HP가 0이 되면 실패.
- 무료 귀항 또는 선언된 60◆ 구조 요청 중 선택.
- 스테이지 선택, 편성, 진행, 천장, 인벤토리와 엔드게임 기록은 즉시 저장.
- 다이아 잔액은 저장하지 않는다.

## 주요 시스템

- 동일 solver 기반 조준 미리보기와 실제 발사.
- 원·캡슐·선분·부위 콜라이더와 swept-circle 충돌.
- 적 공격 카운트와 턴 종료 행동.
- 우정기, 가호, 약점, 부위 파괴, 장애물과 항로 고유 힘장.
- 캐릭터 수집, 레벨, 각성, 중복 조각, 동료도, 장비 유물.
- 가챠 천장, 상점, 설정, 도움말, 일시정지, 결과, 저장 복구.

## 아트 방향

- 승인된 2D 카툰 콘셉트의 짙은 갈색 잉크선, 2~3단 셀 셰이딩, 에게해 네이비·청록·크림·낡은 청동·산호색.
- 1~2등신 둥근 고양이와 정본 얼굴·장비·해부 잠금.
- 세로 720×1280 플레이 캔버스. 웹 카드 대시보드가 아니라 하나의 게임 월드.
- 배경과 충돌 오브젝트, 캐릭터 본체와 FX, 프레임과 텍스트를 분리한다.

## 넣지 않을 것

- 퀴즈, 학습 미션, 스트릭, 학습 복귀, 다이아 획득 CTA 또는 딥링크.
- 별도 공·화살·탄환을 주 발사체로 사용하는 전투.
- 배를 직접 조작하는 전투.
- 외부 CDN·네트워크·사용자 파일·클립보드·학습 카드 접근.
- 참조 게임의 캐릭터·세계관·수치·UI 복제.
