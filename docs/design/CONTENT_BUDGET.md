# 고양이 오디세이 — Content Budget

## 캠페인

- 10항로, 총 43스테이지.
- 일반 항로 7개 × 4스테이지.
- 핵심 항로 3개 × 5스테이지: 외눈박이 동굴, 재앙의 해협, 이타-캣 궁전.
- 모든 스테이지는 데이터 정의이며 스테이지별 전용 TS 코드를 금지한다.

## 전투 콘텐츠

- 플레이 영웅 16종: 정본 캐릭터와 전투 파생형.
- 공용 적 행동군 7종: 돌진, 사격, 방패, 중량, 지원, 분열, 소환.
- 일반 적 외형 20종 이상, 정예 변형 10종.
- 고유 결전 10개, 완전 전용 보스 로직 4개.
- 경기장 셸 18개, 보스 셸 5개.
- 항로 고유 물리 모듈 10개: 파도, 감속, 파괴벽, 바람, 단방향 거울, 위상 포털, 회전 음파, 흡입, 피뢰/금지체, 궁전 종합.
- 목표 리듬은 섬멸 연속을 허용하지 않는다. 외눈박이 동굴은 `섬멸→생존→탈출→섬멸→부위`, 퍼-씨 정원은 `섬멸→생존→탈출→봉인`으로 원전 행동과 승리 조건을 맞춘다.

## 성장·수집

- 레벨, 각성 5단계, 충돌 특성, 우정기, 필살기.
- 유물 32종, 가호 30종, 업적 24종.
- 영웅 도감과 괴수 도감.
- 보스 동료도 1~99.

## 엔드게임

- 신탁탑 30층.
- 폭풍 항로 12노드 주간 시드 런.
- 12마리 중복 금지 3파티 괴수 토벌.
- 영웅별 숙련 과제와 개인 최고 기록.

## QA 하한

- 43개 스테이지 모두 1인·2인·3인 파티로 자동 진입·한 턴·직렬화 스모크. 이 스모크는 실제 승리를 보장하지 않음.
- 제한 턴 내 클리어는 목표별 결정론적 스크립트 또는 기록된 수동 플레이로 별도 검증. 현재 집중 검수 조합은 `MANUAL_CLEAR_AUDIT.md`에 공개.
- 서로 다른 속성이나 특정 영웅 접촉을 요구하는 목표는 솔로/듀오 대체 규칙까지 검증.
- 예상선과 실제 반사 결과 일치.
- 모든 소비 액션 mock 성공·취소·부족·중복 클릭 검증.
- 1280×720, 1920×1080, 720×1280 화면 검수.
- 첫 실행부터 전투·보상·강화·재출항·저장 복구 E2E.
- 플레이어에게 보이는 미지원 효과 0, 목표 설명과 런타임 승리 규칙 일치.
- 모든 스테이지별 배경·벽·프롭 presentation 커버리지. 제작 중 waiver는 authored와 별도 집계하고 1.0 전에 제거.
